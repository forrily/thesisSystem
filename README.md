# thesisSystem


# 技术路线规划
## 后端
- 使用node.js技术。 使用express框架简化开发。
## 前端
* 使用HTML5技术。 使用jQuery、BootStrap框架简化开发。
- 模板引擎
* 使用ejs模板引擎
- 包管理
* 使用Bower管理前端依赖包， 使用npm管理node.js的包。
- 热部署
* 使用supervisor进行代码热部署
- 数据库
* 使用开源的MySQL数据库 和node.js的node-mysql模块
- 项目托管
* 使用GitHub，项目仓库：https://github.com/forrily/thesisSystem

# 安装启动
> npm install 
# 启动服务
> npm start
