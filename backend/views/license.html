技术路线规划
  1. 后端
    2. 使用node.js技术。 使用express框架简化开发。
  2. 前端
    3. 使用HTML5技术。 使用jQuery、BootStrap框架简化开发。
  3. 模板引擎
    4. 使用ejs模板引擎
  4. 包管理
    5. 使用Bower管理前端依赖包， 使用npm管理node.js的包。
  5. 热部署
    6. 使用supervisor进行代码热部署
  6. 数据库
    7. 使用开源的MySQL数据库 和node.js的node-mysql模块
  7. 项目托管
    8. 使用GitHub，项目仓库：https://github.com/forrily

  • 重点介绍
  • SQL
    • 所有功能几乎都是对关系型数据库的操作，所以对SQL应该非常熟悉才行。 我们使用的是MySQL数据库。 使用node-mysql作为驱动进行连接。MySQL数据库的部署， 建议选择阿里云的RDS。 当然你也可以选择其他云数据库，或者自己安装MySQL也行。
  • cookies vs session
    • 如何判断一个用户是否已经登陆了，我们可以使用cookies或者session。 所以，您应该清楚cookies和session的原理，以及他们之间的不同， 并且知道node.js中如何处理cookies或者session的。
